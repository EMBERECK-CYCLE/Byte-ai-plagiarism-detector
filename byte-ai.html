What is Byte AI?

Byte AI is a project made for the 2024-2025 International Science Fair. It was created by users HeartyDarty and EMBERECK-CYCLE on Github. Byte AI is an AI detector that was created to aid teachers in spotting AI made works being submitted by students, however will most likely not reach the stage of being able to be used in professional/academic spaces. We started learning code just for this project, and have worked long and hard to make this project. This project may not receive updates, and if it does they will be far and few.
